# SCT_CS_01

Caesar Cipher
A simple Python implementation of the Caesar Cipher encryption technique.
Description
The Caesar Cipher shifts each letter in the alphabet by a specified number of positions. For example, with shift 3: 'A' → 'D', 'B' → 'E'.
Requirements

Python 3.x

Usage
bashpython caesar_cipher.py
Choose from the menu:

Encrypt a message
Decrypt a message
Exit

Example
Message: Hello World
Shift: 3
Encrypted: Khoor Zruog
Features

Encrypt and decrypt messages with shift values 1-25
Preserves spaces, punctuation, and case
Interactive command-line interface
